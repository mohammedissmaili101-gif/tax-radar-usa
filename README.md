# 🎯 TaxRadar.io
### The Smartest Freelance Tax Forecast Engine for 2026

**TaxRadar** is a high-performance, privacy-first tax simulation tool built to solve the biggest problem for US-based freelancers: **Understanding true net income after Federal, FICA, and State taxes.**

## 🚀 Key Features
- **Instant Calculations:** Real-time net pay breakdown.
- **State Duel:** Compare taxes between high-tax states (California, New York) and 0% tax havens (Texas, Florida).
- **Privacy by Design:** No database, no cookies, no tracking. Calculations happen 100% on the client side.
- **Clean UI:** Premium Dark Mode optimized for high-end professional users.

## 🛠️ Tech Stack
- **Frontend:** HTML5 / CSS3 (Inter & JetBrains Mono fonts).
- **Logic:** Vanilla JavaScript (High-precision tax bracket math).
- **Deployment:** Vercel / GitHub Pages.

## 📈 SEO Optimized
Built with **JSON-LD Schema** to target Google's Featured Snippets and high-intent keywords like `freelance tax calculator 2026`.
